## BuilderMania (Next.js)

Application web Next.js (App Router) avec composants UI (style shadcn/ui), Tailwind CSS v4, et intégration Vercel Analytics.

### Prérequis

- Node.js (version moderne)
- pnpm

### Installation

```bash
pnpm install
```

### Lancer en développement

```bash
pnpm dev
```

### Build & démarrage

```bash
pnpm build
pnpm start
```

### Lint

```bash
pnpm lint
```

### Documentation

- `docs/public-api.md` : APIs publiques (fonctions, composants, exports)
- `docs/routes.md` : routes/pages publiques
- `docs/styling.md` : thème, Tailwind, conventions UI
